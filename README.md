# 🤖 AI Learning Assistant

An AI-powered learning assistant built using **Streamlit** and **Groq Llama 3.3 70B**. It provides personalized explanations based on a student's grade and learning style, supports PDF-based question answering, and tracks learning progress during the session.

## ✨ Features

- 📚 Personalized AI tutoring
- 📄 Ask questions from uploaded PDFs
- 👨‍🎓 Student profile customization
- 📊 Learning progress tracking
- 💬 Interactive chat interface
- 🎨 Clean and responsive UI

## 🛠️ Tech Stack

- Python
- Streamlit
- Groq API
- Llama 3.3 70B Versatile
- PyPDF
- python-dotenv

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/ai-learning-assistant.git
cd ai-learning-assistant
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Create a `.env` file

```env
GROQ_API_KEY=your_api_key
```

4. Run the application

```bash
streamlit run app.py
```
